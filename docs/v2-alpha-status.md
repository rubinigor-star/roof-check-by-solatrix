# Roof Check v2 Alpha

Status: Alpha preview.

Ready for testing:

- Hebrew customer flow.
- Address and monthly electricity bill input.
- Roof type selection.
- Mock map layer prepared for future real map provider.
- Interactive roof marking on the mock map.
- Multiple roof surfaces for complex roofs.
- Obstacle selection.
- AI-style analysis screen.
- Mixed tariff calculation: self consumption at 0.64 NIS/kWh and export at 0.48 NIS/kWh.
- Effective tariff output.
- Result cards.
- Savings/export mix graphic.
- Printable PDF report window.
- Official Solatrix logo.

Not connected yet:

- Real satellite map provider.
- Real address geocoding.
- Real polygon area from map coordinates.
- WhatsApp/email delivery automation.

Next Alpha improvements:

- Solar Score block in the live result screen.
- Monthly production chart.
- 3 system scenarios: conservative, recommended, maximum.
- 6 page PDF report layout.
- QR code for WhatsApp.

Testing checklist:

1. Open the site on mobile.
2. Start roof check.
3. Enter address and monthly bill.
4. Pick roof type.
5. Mark roof on the map.
6. Add obstacles.
7. Review result.
8. Generate PDF.
9. Confirm that all customer-facing text is Hebrew.
